# Renew GHL Contact

Refresh and renew a GoHighLevel contact for an urban renewal project. When given contact information or a contact ID:

1. Review the existing contact data provided
2. Identify stale or outdated fields (last activity date, tags, pipeline stage)
3. Suggest or apply updates to:
   - Contact tags (add `urban-renewal`, remove obsolete tags)
   - Pipeline stage (move to appropriate stage based on project status)
   - Custom fields relevant to urban project tracking
   - Notes with a renewal timestamp
4. Generate a summary of changes made or recommended

If the user provides `$ARGUMENTS`, treat it as the contact name, ID, or project name to renew.

Always confirm the contact and project context before making changes.
