# Renew GHL Opportunity

Reactivate and refresh a GoHighLevel opportunity tied to an urban renewal project.

Steps:
1. Review the opportunity details provided (name, stage, value, last activity)
2. Assess renewal viability based on:
   - Days since last activity
   - Current pipeline stage
   - Associated contact engagement
3. Recommend or apply:
   - Updated close date
   - Revised opportunity value if applicable
   - Stage progression or re-entry point
   - Follow-up task creation
   - Note with renewal rationale and timestamp
4. Output a renewal action summary

If `$ARGUMENTS` is provided, use it as the opportunity name, ID, or search term.
